    Use the man -k to search the keyword you want to find (Ex: man -k keyword)
after that there are a number inside the () that stand for each section in linux

1: command
2: system call
3: library's function
4: device's file
5: file format
6: game
7: other various things
8: system management's command
9: Kernel Ruhen

    User can also use the command: man section'snumber name, to search
    (Ex: man 1 cat)

    Use this command to search section's number and the path:
    man -wa name (Ex: man -wa cat)
